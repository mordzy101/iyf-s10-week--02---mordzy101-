## Project Description
This project is a simple personal portfolio website. It introduces who I am, my hobbies and interests, a photo gallery, and my contact information.

The goal of this project was to practice using HTML and CSS to build a clean and responsive web page.

## Technologies Used
- HTML5
- CSS3
- Google Fonts
- Flexbox
- CSS Grid

## Features
- Responsive navigation bar
- About section with a profile photo
- Hobbies and interests cards
- Photo gallery using CSS Grid
- Contact section with email and favorite website link
- Mobile-friendly layout

## How to Run
1. Clone this repository.
2. Open the project folder.
3. Open `index.html` in your browser.

## Lessons Learned
While building this project, I learned how to connect an HTML file to a CSS file, use Google Fonts, style a web page with CSS, and create layouts using Flexbox and CSS Grid.

I also learned how to make a website responsive so it looks better on both small and large screens.

## Challenges Faced
One challenge was linking the images correctly. I solved this by placing the gallery photos inside an `images` folder and updating the image paths in `index.html`.

Another challenge was keeping the design simple while still making it look good and responsive.
